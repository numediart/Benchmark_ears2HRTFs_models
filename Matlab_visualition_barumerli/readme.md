# Visualisation of localisation test simulation #

## Setup ##
You need to download and install the Auditory Modeling Toolbox: [link](https://www.amtoolbox.org/) 
You can then create a folder to store all the sofa files HRTFs (contain in Le-roux-1D.zip for example), you will have then to modify "path_norm" in the matrix_plot_results.m code.
## Visualisation ##
You just have to run matrix_plot_result.m as standalone code.

Once run, the code should plot you similar results as:
